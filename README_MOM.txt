1. Open terminal
2. Navigate to folder containing MedianOMedians.py and any other input files
3. Enter the following into the terminal: 
python MedianOMedians.py input_filename k i
    input_filename: the name/path of the input file
    k: the block size
    i: the ith smallest element to choose

Example: python MedianOMedians.py input.txt 5 15
